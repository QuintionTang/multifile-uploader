# multifile uploader

多文件上传

--

### Quick Start

-   Install latest node.js: https://nodejs.org​
-   Install latest yarn package manager: https://yarnpkg.com/​
-   Install node modules by running terminal command `yarn install`

#### Run the service

-   Run `cd api-service`
-   Run `yarn dev`

### Run the app

-   Run `yarn start`
-   Or build production app `yarn build`
